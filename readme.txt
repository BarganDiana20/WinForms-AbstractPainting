AbstractPainting

Proiect universitar - Aplicație Windows Forms: o aplicație capabilă să deseneze „picturi abstracte” folosind forme geometrice de bază, linii și curbe Bezier, selectate de utilizator. Pictura poate fi apoi colorată de utilizator, completând spațiile dintre linii.
